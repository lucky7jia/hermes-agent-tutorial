模块 10 / 10

定时任务 (Cron Jobs)

Agent 在没有人操作的情况下自动完成工作——定时汇总、监控、报告。

预计阅读时间：8 分钟

1 Cron 调度器

Cron 让 Agent 在指定时间自动运行。每个 Cron Job 是一个独立的 Agent 会话——可以配置模型、Skills、工作目录和投递目标。

hermes cron list         # 列出所有任务
hermes cron create SCHED   # 创建任务
hermes cron edit ID      # 编辑
hermes cron pause/resume ID # 暂停/恢复
hermes cron run ID       # 立即触发
hermes cron remove ID     # 删除
2 调度表达式
格式示例含义
时长30m, 2h每 30 分钟 / 每 2 小时
everyevery monday 9am每周一早上 9 点
5 字段 Cron0 9 * * *每天早上 9 点
ISO 时间戳2026-06-01T09:00一次性定时
3 两种运行模式

🧠 LLM 模式（默认）

Agent 运行 prompt，可以推理、调工具、生成报告。适合需要判断力的任务。

📜 Script 模式 (no_agent=true)

只运行脚本，stdout 作为消息投递。零 token 消耗。适合纯数据收集、监控。

4 任务链

通过 context_from 字段，可以把前一个 Job 的输出注入到下一个 Job 的上下文中，实现任务流水线：
