# 模块 8：委托与多智能体 (Delegation)

> 预计阅读时间：8 分钟 · 前置：[模块 7](../07-gateway/)

派生子 Agent 并行工作——任务拆解、各自攻坚。

> 💡 推荐：浏览器打开 `index.html` 获得最佳体验（深色主题 + Mermaid 图表）。

---

## 1. delegate_task

父 Agent 派生子 Agent 处理独立子任务。每个子 Agent 有隔离的上下文和终端。

## 2. 两种模式

| 模式 | 场景 | 用法 |
|------|------|------|
| 单任务 | 一个复杂子任务 | `delegate_task(goal="...")` |
| 批量并行 | 多个独立任务 | `delegate_task(tasks=[{...},{...}])` 最多 3 并发 |

## 3. delegate_task vs 手动 spawn

| 维度 | delegate_task | 手动 spawn |
|------|-------------|-----------|
| 隔离度 | 独立上下文 | 完全独立进程 |
| 时长 | 分钟级 | 小时/天级 |
| 适用 | 快速并行任务 | 长期自主任务 |
