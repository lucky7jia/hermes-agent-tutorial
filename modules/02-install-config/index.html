1 安装 Hermes（全平台）
⚠️ 已废弃：pip install v0.17.0 起不再推荐 pip 安装。官方唯一推荐方式为 curl 一键脚本（Linux/macOS/WSL2/Termux）或 PowerShell 脚本（原生 Windows）。如你在任何地方看到 pip install hermes-agent，该文档已过时。

🐧 Linux / macOS / WSL2

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

🪟 Windows 原生（PowerShell）v0.17 完善

iex (irm https://hermes-agent.nousresearch.com/install.ps1)

安装器自动处理：uv、Python 3.11、Node.js、ripgrep、ffmpeg、便携 Git Bash（MinGit，解压到 %LOCALAPPDATA%\hermes\git，无需管理员权限，与系统 Git 完全隔离）。若已有 Git 则自动复用。

📱 Android / Termux v0.17 已验证

官方验证路径见 Termux 安装指南。注意：Termux 下安装的是精简版（.[termux] extras），完整版因 Android 音频依赖问题暂不可用。

安装后：source ~/.bashrc（或 ~/.zshrc）重新加载 shell，然后直接 hermes 开始使用。

hermes setup gateway      # 只配平台接入
hermes setup --portal      # 🚀 OAuth 一条龙（见 §6）

3选模型从 Provider 的模型列表中选默认模型

3 config.yaml 深度解析
hermes config           # 查看
hermes config edit      # 编辑器打开
hermes config set section.key value # 命令行改单项
段作用关键字段
model模型配置（必填）default, provider, base_url, context_length
agentAgent 行为max_turns(90), tool_use_enforcement
terminal终端后端 v0.17backend(local/docker/ssh/singularity/modal/daytona), timeout
compression上下文压缩threshold(0.50), target_ratio(0.20)
memory持久记忆memory_enabled, user_profile_enabled, provider
approvals命令审批mode(manual/smart/off)
delegation子代理model, max_iterations
curator v0.17Skill 自动维护enabled, interval_hours, stale_after_days
checkpoints文件快照enabled, max_snapshots
⚠ 改配置后需要重启：Gateway /restart，终端退出重进。System Prompt 在会话启动时构建，改配置不会热加载。
4 .env 密钥管理
OPENROUTER_API_KEY=YOUR_OPENROUTER_KEY
ANTHROPIC_API_KEY=YOUR_ANTHROPIC_KEY
HERMES_HOME=/home/user/.hermes  # 可选：自定义数据目录

密钥放 .env，不入 git。环境变量优先级高于 config.yaml 中的 api_key 字段。

6 Nous Portal 一体化工具网关 v0.17 新增
一条命令，搞定模型 + 搜索 + 生图 + TTS + 云浏览器

如果你不想分别申请 5 个 API Key（模型、搜索、生图、TTS、浏览器），Nous Portal 一个订阅全包：

  • 300+ 模型——/model 任意切换
  • Tool Gateway——Web 搜索（Firecrawl）、图像生成（FAL）、TTS（OpenAI）、云浏览器（Browser Use）
hermes setup --portal  # OAuth 登录 → 自动配好所有工具
hermes portal info    # 查看当前工具网关连接状态

你仍然可以随时用自己的 Key 逐工具配置——网关是按后端可选的，不是全有或全无。

10 常见疑问
Q: pip install 还能用吗？
⚠️ 已废弃。 v0.17.0 不再推荐 pip 安装。唯一推荐方式：curl 脚本（Linux/macOS/WSL2/Termux）或 PowerShell（Windows）。pip 路径可能缺少运行时依赖、工具链集成不完整。
Q: Windows 上能跑吗？需要 WSL 吗？
不需要 WSL。 Hermes v0.17 原生支持 Windows（PowerShell）。安装器自动装 MinGit——一个独立的 Git Bash，不与系统 Git 冲突。CLI、Gateway、TUI、所有工具全部原生可用。
Q: Nous Portal 和 OpenRouter 什么区别？
OpenRouter 是纯模型 API 聚合。Nous Portal 除了 300+ 模型，还提供一体化工具网关——Web 搜索、生图、TTS、云浏览器一个订阅全包，不用分别申请 5 个 Key。
